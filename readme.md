\# Mars Colony ABS



Agent-based simulation for the Mars Colony collaborative task.



\## Current structure



\- `interface.py` - GUI entry point

\- `modules/` - simulation modules

\- `Documentation/` - task documents, roadmaps, and supporting materials



\## Notes



This repository contains an in-progress prototype of a Mars Colony agent-based simulation involving team cognition, DIK transformations, communication, construction, and experiment configuration.



## Quick launch (double-click)

- Use `launch_interface.sh` from the repository root to open the simulation interface directly.
- On Linux desktop environments, you can double-click the script and choose **Run**.
- On macOS, run `chmod +x launch_interface.sh` once, then double-click in Finder (or run from Terminal).
